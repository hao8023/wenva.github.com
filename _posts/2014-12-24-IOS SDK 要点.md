# IOS SDK 要点
December 24, 2014

## XIB
* XIB 中的File Owner是指谁加载了此XIB文件, 即那个对象调用loadNibNamed或initWithNibName方法


## UI SDK
* UINavigationBar 设置背景图时（setBackgroundImage）必须保证图片高度为44（或@2x）
* UIImage可以设置拼接UIImageResizingModeTile和拉伸UIImageResizingModeStretch 2种模式
